Installing Anglican

Mac / Linux / Unix

Download and run anglican (right click and save it to a file called anglican). That’s it.

Make sure to chmod anglican so that it is runable and to put in somewhere on your path (i.e. chmod 755 anglican; export PATH=${PATH}:.). Note that this script will create and download jar files to ~/.anglican. Feel free to read it to ensure that it doesn’t do anything evil.

Note – Anglican invoked with no command line arguments expects source to be piped into into it on stdin. The first time you run anglican please run anglican --help.

This script requires internet connectivity, Java, and curl.

Windows / Other Operating Systems / Advanced

Anglican is implemented in Clojure. This means that there are three ways of downloading, installing, and running Anglican.

1) Download an executable jar

The current release version can be downloaded from here.

Running 

java -jar anglican-afb5bd8412aa1d5d84df764de8caa3a3d38c35ab-master.jar --help

prints usage instructions.

2) Use leinegen to create a Clojure project with Anglican dependency

Edit project.clj and add the :jvm-opts, :main, and :dependencies lines to your project.

(defproject your-project-name "your-version-string"
  :jvm-opts [ "-Xms16G" "-Xmx16G"]
  :dependencies [[org.clojure/clojure "1.4.0"]
                 [anglican "0.1.0-SNAPSHOT"]]
  :main anglican.main)

Then run 

lein run -m main --help

from within the project. This runs the leinegen project and automatically invokes anglican.main and prints out command line arguments.

3) Clone and fork the Anglican git repo (source access)

The git repository is here but currently private. Please email to request access.

Development Release

The current development release can be downloaded from here. To run the development version of Anglican please invoke it using the Java command line form.
